This example shows how to integrate GeoJSON layers and style them.

This can be achieved by using a GeoJSON file and a styling file containing the associated features styles. The GeoJSON file contains the geospatial topology using the Javascript Object Notation. It handles geometries such as Points, Linestrings, Polygon, MultiLineString and MultiPolygon (See GeoJSON documentation). The styling file is a JSON file containing styling information about the features included in the GeoJSON file. Syling properties and options come from OpenLayers 3 Feature styles (learn more about feature styles on Openlayers 3 documentation).

Just enter your own GeoJSON file and associated styling file URLs to visualize them on this map.
